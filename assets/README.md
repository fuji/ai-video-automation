# Assets フォルダ

動画生成で使用するアセットを格納します。

## BGM (`bgm/`)

ニュースのムードに合わせたBGMを配置してください。

| ファイル名 | 用途 | 推奨 |
|-----------|------|------|
| `upbeat.mp3` | 明るい・楽しいニュース | テンポ良い、ポジティブな曲 |
| `emotional.mp3` | 感動系・ほっこり | ピアノ、ストリングス系 |
| `quirky.mp3` | おもしろ・奇妙なニュース | コミカル、軽快な曲 |
| `dramatic.mp3` | 衝撃・驚きのニュース | テンション高め |
| `neutral.mp3` | 一般的なニュース | ニュートラルなBGM |

### おすすめフリーBGMサイト
- [Pixabay Music](https://pixabay.com/music/) - 完全無料、クレジット不要
- [FreePD](https://freepd.com/) - パブリックドメイン
- [Dova-Syndrome](https://dova-s.jp/) - 日本のフリーBGMサイト
